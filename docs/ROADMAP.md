# Roadmap

## Phase 1 — Two bags, one camera, no AI

- Capture snapshot from Frigate or file
- Define ROI for bag1 and bag2
- Calculate green coverage
- Calculate yellow/brown coverage
- Publish metrics to MQTT
- Home Assistant dashboard

## Phase 2 — Sensors

- ESP32
- SHT31 or BME280
- BH1750 lux sensor
- Moisture sensor per bag
- MQTT integration

## Phase 3 — Rules Engine

- Watering recommendations
- Heat stress detection
- Growth stagnation detection
- Yellowing alerts

## Phase 4 — Historical Analytics

- Daily growth trend
- Weekly trend
- Timelapse generation

## Phase 5 — Hailo-8 Integration

- Remote inference worker
- Plant segmentation
- Weed detection
- Disease detection

## Phase 6 — Scale to 12 bags

- Multi-camera support
- 12 ROI configuration
- Crop-specific profiles

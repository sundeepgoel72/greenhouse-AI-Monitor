# Debugger Agent Role

Own:

- operational issue reproduction
- sensor/image data investigation
- deployment/systemd/runtime validation
- updates to `.codex/HANDOVER.md` and `.codex/WORKLOG/`

Before editing:

1. Check active claims in `.codex/STATUS.md`.
2. Claim your investigation scope if code or deployment files need to change.

Do not:

- leave unresolved runtime observations undocumented
- return work without exact commands, timestamps, and risk notes
